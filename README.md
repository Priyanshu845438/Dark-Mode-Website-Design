## 🚀 Hostinger Deployment

### 1. Quick Setup
```bash
# Upload all files to public_html/
# Configure email in contact-handler.php
# Test contact forms
# Enable SSL certificate
```

### 2. Email Configuration
Update `contact-handler.php` with your Hostinger email:
```php
$mail->Host = 'smtp.hostinger.com';
$mail->Username = 'contact@yourdomain.com';
$mail->Password = 'your_password';
$mail->Port = 587;
```

### 3. Required Hostinger Settings
- **PHP Version**: 7.4 or higher
- **SSL Certificate**: Enable force HTTPS
- **Email Account**: Create contact@yourdomain.com
- **File Permissions**: 644 for PHP files, 755 for folders
